# Tripod.geometricPi1OverQbar

The profinite etale fundamental group of the thrice-punctured projective line
`P^1_{Qbar} \ {0, 1, infinity}` with base point over
`Qbar = AlgebraicClosure Q`.

By the base-change theorem for etale fundamental groups (SGA1 X.1.8), this is
isomorphic to `Tripod.geometricPi1OverC` via the embedding `Qbar -> C`.
